About

This blog is about the development of the moxie processor core, an open source soft-core processor, and related tooling.

Moxie is a general purpose bi-endian load-store processor, with sixteen 32-bit general purpose registers and a comprehensive ISA consisting of two-operand variable width instructions. There are moxie implementations that run on both Alera and Xilinx FPGA architectures, and a complete GNU toolchain for C/C++ programming.

HDL source for the core is available here http://www.github.com/atgreen/moxie-cores.

The author of this blog is http://moxielogic.org/blog/pages/author.html.
