Toolchain

The toolchain for the moxie architecture consists of the following tools: - the GNU assembler, linker and binary utilities - GCC, the GNU Compiler Collection - GDB, the GNU Debugger, including a port of the GDB simulator to the moxie architecture - QEMU, the Open Source Processor Emulator - Newlib, a C library for bare-metal embedded software development

Supported target triplets include moxie-elf, moxie-rtems (for RTEMS application development), and moxiebox (a special purpose configuration for the moxiebox bitcoin oracle project).
